Модуль управления звуковыми эффектами мессенджера.

Стек: audioplayers.

Структура
sound_service.dart: Синглтон для управления аудио-сессиями.

sound_type.dart: Перечисление (enum) доступных звуков.

Функционал
Инициализация: Загрузка файлов в оперативную память через AudioCache при запуске.

Воспроизведение: Метод play(SoundType type) для проигрывания эффектов (отправка, получение, клики).

Оптимизация: Минимальные задержки за счет предварительного кэширования.

Использование
Dart
````
// Входная точка
await SoundService.instance.init();

// Триггер звука
SoundService.instance.play(SoundType.messageSent);
````
Файлы
Звуки должны находиться в assets/sounds/ и быть прописаны (Если я не забыл) в pubspec.yaml
