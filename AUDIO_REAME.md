Модуль Sound (lib/sound)
Модуль обеспечивает воспроизведение системных звуков (уведомления, отправка сообщений) с использованием библиотеки audioplayers.

Основные возможности
Кэширование: Предзагрузка аудиофайлов при старте приложения для исключения задержек.

Событийная модель: Воспроизведение привязано к конкретным действиям (отправка, получение, ошибки).

Микширование: Поддержка наложения звуков без прерывания текущих потоков.

Ключевые методы
init(): Инициализация AudioCache и регистрация путей к ассетам.

play(SoundEvent event): Запуск проигрывания звука, соответствующего событию.

stopAll(): Принудительная остановка всех активных звуковых потоков.

Конфигурация
Аудиофайлы хранятся в assets/sounds/. Основные триггеры:

message_sent: Звук подтверждения отправки.

message_received: Уведомление о входящем сообщении.

Интеграция

~~~
// Инициализация
SoundService.instance.init();

// Вызов звука
SoundService.instance.play(SoundEvent.sent);
~~~
